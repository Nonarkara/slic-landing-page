# SLIC Index V1 — Where It All Started

**The original provocation.** First published on LinkedIn in 2025, this is the version that sparked a movement. 103 cities ranked transparently — every score traceable, no hidden formulas, no prestige bias.

> "We stopped ranking. We started telling the truth."

## What This Is

The Smart and Liveable Cities Index (SLIC) V1 was the first public attempt to rank cities by what actually matters to residents — not expats, not tourists, not investors. Disposable income after rent. Working hours. Tolerance. Cultural diversity.
